# 内容维护清单

## 每次新增文章必须完成

- [ ] 确认文章编号没有重复。
- [ ] 文章放入正确中文分类目录。
- [ ] 文章隐藏元信息块字段完整，且用 HTML 注释隐藏，不在预览页展示。
- [ ] 正文不重复展示文章一级标题，从核心观点、封面图或正文小标题直接开始。
- [ ] 封面放入 `08-素材库/图片/文章封面/`，正文插图放入 `08-素材库/图片/正文插图/`。
- [ ] 图片文件名与文章编号、文章标题一致。
- [ ] 更新 `README.md`。
- [ ] 更新 `00-知识库导航/知识库首页.md`。
- [ ] 更新 `00-知识库导航/知识库总索引.md`。
- [ ] 更新 `00-知识库导航/分类索引.md`。
- [ ] 更新 `00-知识库导航/系列索引.md`。
- [ ] 更新 `00-知识库导航/标签索引.md`。
- [ ] 更新对应分类目录的 `README.md`。
- [ ] 更新 `07-资料与流程/发布记录.md`。
- [ ] 检查本地链接、图片和代码块。
- [ ] 如涉及最新事实，已核验来源并写清截至日期。

## 每次更新旧文章必须完成

- [ ] 保留原文章编号，不重编号。
- [ ] 如修改标题，同步更新文件名、图片名、隐藏元信息、索引和所有内部链接。
- [ ] 如移动分类，同步更新主分类、分类索引、总索引和旧链接说明。
- [ ] 如替换图片，删除未引用旧图或移入素材库正确分类。
- [ ] 更新隐藏元信息中的 `updated` 与 `source_status`。
- [ ] 在发布记录中记录本次变更。

## 建议每月复查

- [ ] 过时模型名、产品名、价格、政策。
- [ ] 外部链接可访问性。
- [ ] 文章分类是否需要拆分新专题。
- [ ] 素材库是否存在未使用图片。
- [ ] 文章生成流程是否需要继续沉淀到 `article-wechat` Skill 或 `AGENTS.md`。
