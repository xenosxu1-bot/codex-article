# 内容维护清单

> 更新时间：2026-07-15

## 当前维护规则

- [x] 正式文章正文不展示顶部元信息。
- [x] 正式文章正文不重复一级标题。
- [x] 文章元信息外置到 `文章资产登记表.md`。
- [x] 文章封面与正文插图保存在素材库。
- [x] 已建立总索引、分类索引、标签索引、系列索引。
- [x] 后续文章生成方案已沉淀为固定流程。

## 新文章入库检查

- [ ] 文件名包含连续编号和中文标题。
- [ ] 正文不含顶部 YAML / HTML 注释元信息。
- [ ] 正文不重复 `# 标题`。
- [ ] 图片链接可打开。
- [ ] `文章资产登记表.md` 已补全编号、标题、分类、标签、状态、封面。
- [ ] 总索引、分类索引、标签索引、系列索引已更新。
- [ ] 对应分类 README 已更新。
- [ ] 发布记录已更新。
- [ ] 涉及最新事实的内容已核验。

## 导航索引检查

- [ ] 分类是否唯一，并与文章所在目录一致。
- [ ] 系列是否代表阅读路径，而不是重复分类名。
- [ ] 标签是否用于检索，每篇控制在 3-5 个。
- [ ] 首页、总索引、分类索引、系列索引、标签索引是否与文章资产登记表一致。
- [ ] 读者入口和维护入口是否分离，避免把后台登记信息放到正文或首页主阅读区。
