- `已发布`：已在对外平台上线，在《发布记录》备注平台和日期。
- `待上线`：正文已入库并通过发布前检查，尚无可核实的对外上线记录。
- `修正中`：已发现正文、图片或实操问题，暂不建议上线。
- `暂不发布`、`已下架`：保留编号与追溯记录，不触发自动重编号。

- 仓库 Markdown、脚本和流程文档统一使用 UTF-8 编码与 LF 换行，避免 GitHub、Windows 编辑器和公众号转换工具之间出现显示差异。

- 如果在 Windows 终端看到中文路径乱码，优先判断为终端显示编码问题；以文件实际 UTF-8 内容、GitHub 页面和本地内容校验为准。
