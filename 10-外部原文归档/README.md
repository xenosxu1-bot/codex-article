## 文件约定

- `source.html`：下载时的原始 HTML 快照，是内容追溯的唯一原文。
- `content.md`：由原始 HTML 自动转换得到的 Markdown，只转换格式，不人工改写正文。
- `content.txt`：规范化纯文本，仅供检索、指纹计算和重复度检查。
- `metadata.json`：来源、权属、抓取日期、哈希和去重规则。

## 新增外部原文

1. 先确认内容权属与公开归档范围。
2. 使用 `node 09-工具脚本/归档外部原文.mjs` 保存原始 HTML 与机械转换文本。
3. 运行 `node 09-工具脚本/重建内容库索引.mjs` 更新统一内容库索引。
4. 新建正式文章前运行 `node 09-工具脚本/检查内容重复度.mjs --candidate <候选文件>`。重复度必须低于 **50%**。
