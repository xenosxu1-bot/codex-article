# 给AI新手的100个神级Prompt模板

先记住Prompt万能公式
你可以先记成一句话：
角色 + 目标 + 背景 + 要求 + 输出格式

🧠 第一组：AI通识类 Prompt
适合刚开始学AI的人。 先把认知搭起来，再谈提效。
1. 你是一位AI入门老师，请用最通俗的方式解释“什么是大模型”，要求像讲给高中生听。

2. 请用“生活比喻”的方式解释AI、机器学习、深度学习三者的区别。

3. 请告诉我，ChatGPT、Claude、通义千问、文心一言各自适合做什么。

4. 请用“人话”解释Prompt是什么意思，别用术语。

5. 请总结AI最常见的5个误区，并告诉我为什么会误解。

6. 请用一张表帮我梳理“AI能做什么、不能做什么”。

7. 请解释“AI为什么会胡说”，要求简单易懂。

8. 请用案例讲清楚“大模型为什么像在预测下一个词”。

9. 请告诉我，普通人学习AI最合理的顺序是什么。

10. 请帮我做一份“AI入门第一周学习清单”，每天1小时即可。

先理解AI，再使用AI。先建立认知，再追求效率。

✍️ 第二组：写作创作类 Prompt
这一组，最适合自媒体、文案、公众号、短视频脚本。
1. 你是一位10万+公众号主编，请帮我写一个爆款开头，主题是“AI入门第一周学什么”。

2. 请围绕“零基础学AI”写3个不同风格的标题，分别偏焦虑、偏成长、偏实用。

3. 请把这段文字改写得更像人说话，去掉AI味，保留原意。

4. 请帮我把这篇文章改成公众号风格，要求每段不超过5行。

5. 请根据主题“100个神级Prompt模板”写一篇适合小红书的笔记。

6. 请把以下内容改成适合朋友圈发布的短文案，语气自然一点。

7. 请帮我写一段短视频口播稿，时长控制在15秒内。

8. 请为这篇文章生成3个不同版本的结尾，要求有互动感。

9. 请把下面这段内容改成“故事开头+情绪共振+认知升级”的爆文结构。

10. 请把这篇文章压缩成一页海报文案，要求能直接排版。

🧾 第三组：总结提炼类 Prompt
AI最适合帮你做的事之一，就是“压缩信息”。
1. 请用3句话总结下面这篇文章的核心观点。

2. 请把这份长文总结成“重点版”，并按一级、二级、三级要点整理。

3. 请提炼这段内容中的5个关键词，并解释每个关键词的含义。

4. 请把这篇文章整理成“适合收藏”的笔记格式。

5. 请帮我把以下会议纪要提炼成行动清单。

6. 请把这份报告总结成“老板一眼能看懂”的版本。

7. 请将这篇长文拆成“结论、原因、建议”三部分。

8. 请把这段内容浓缩成100字以内。

9. 请提取这篇文章中最有传播性的3句话。

10. 请把下面内容整理成表格，列出“问题、原因、解决方案”。

🌍 第四组：翻译与改写类 Prompt
翻译不是简单换语言。 更好的翻译，是“换成目标读者能懂的话”。
1. 请把这段英文翻译成自然的中文，不要直译。

2. 请把这段中文翻译成适合海外社媒发布的英文。

3. 请把这段专业英文改成适合小白看的中文。

5. 请把这段内容改写成更口语化、更像真人表达的版本。

6. 请把这段文字改成更适合公众号发布的语气。

🧩 这100个Prompt，最正确的用法不是“背”
而是改。
你只要记住三件事：
1. 先给角色。比如：你是一位主编、老师、产品经理、程序员。
2. 先给目标。比如：写文案、总结、翻译、分析、做PPT。
3. 再给限制。比如：口语化、每段不超过5行、输出表格、适合小白。
Prompt写得好，不是因为你会说术语。而是因为你会把任务说清楚。

📎 给新手的避坑提醒
很多人学Prompt，最容易犯三个错：
第一，太笼统。只说“帮我写一下”，AI只能给你泛泛的答案。
第二，要求太多但没结构。一口气塞10个要求，AI也会乱。
第三，只收藏，不实操。看懂不等于会用。 会复制，不等于会改。
🚀 最后给你一个最实用的起点
如果你现在还是零基础。 别急着研究所有玩法。
先从这3个模板开始：
模板1：写作类你是一位[角色]，请帮我写一篇关于[主题]的内容，要求[风格]，输出[格式]。
模板2：总结类请把以下内容总结成[目标]，要求提炼重点，输出为[格式]。
模板3：执行类请根据以下任务，帮我拆解成可执行步骤，并给出每一步的注意事项。
只要你能把这3类模板用熟。 AI就已经开始为你工作了。
