# 图片素材

本目录只保留正式文章实际引用的图片。

## 分类

| 目录 | 内容 | 使用场景 |
| --- | --- | --- |
| `文章封面/` | 文章封面图 | 隐藏元信息块 的 `cover` 字段，以及正文开头复用的封面图 |
| `正文插图/` | 正文过程图、流程图、截图、说明图 | Markdown 正文中的图片语法 |

## 维护规则

1. 新增封面图放入 `文章封面/`。
2. 新增正文配图放入 `正文插图/`。
3. 图片文件名使用“文章编号-中文文章标题-用途”的中文命名：封面为 `35-中文文章标题-封面.jpg`，正文配图为 `35-中文文章标题-正文插图01.png`。标题必须与文章 隐藏元信息块 的 `title` 一致。
4. 删除图片前先确认没有任何正式文章引用。
5. 移动图片后必须同步更新文章中的相对路径，例如把正文图片路径改成 `../08-素材库/图片/正文插图/35-inline-01.png`。
